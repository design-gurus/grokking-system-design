# Resources

A curated list of useful resources for system design interview prep. Pull requests with good additions are welcome (see [CONTRIBUTING](CONTRIBUTING.md)).

## Courses (DesignGurus)

Different courses fit different stages. Pick by where you are:

| Course | Who it is for |
|--------|---------------|
| [Grokking System Design Fundamentals](https://www.designgurus.io/course/grokking-system-design-fundamentals) | New to system design. Learn the core building blocks first. |
| [Grokking the System Design Interview](https://www.designgurus.io/course/grokking-the-system-design-interview) | The core interview prep course, by Arslan Ahmad. Start here for most people. |
| [System Design Interview Crash Course](https://www.designgurus.io/course/system-design-interview-crash-course) | Short on time. A fast, focused path before an interview. |
| [Advanced System Design Interview, Volume II](https://www.designgurus.io/course/grokking-system-design-interview-ii) | Senior and advanced. Harder problems and deeper dives. |

## How to use this list

Start with the [interview framework](cheat-sheets/interview-framework.md) and the core [patterns](patterns/), then practice with the [questions](questions/). Use the free blogs when you want more depth, and the courses when you want the full guided path with diagrams and worked solutions.
